---

### Описание 

**Название:** Приложение для генерации подарочных карт.
**Назначение:** быстро создавать карты подарков для Telegram с указанием цены в USD, цене в звездах, цене в TON, названием подарка, временем действия и картинкой

---

### Начало работы

1. Откройте бота
2. Нажмите «Старт»
3. Заполните поля 
4. Нажмите кнопку «Сгенерировать карту»
5. Посмотрите результат в блоке «Результат»

---

### Общая рекомендация
- Подготовьте URL изображение ( прямая ссылка на картинку: .jpg / .png)
- Рекомендуется использовать изображения с прозрачным фоном или с минимальным фоном для лучшего результата
- Цвета указывать в Hex-формате
- Все числовые поля вводите без лишних символов (использовать только цифры)

---

### Описание полей интерфейса
**Основной цвет:** основной фон карточки. Указывать шестнадцатеричный код цвета.
**Вторичный цвет:** дополнительный цвет для оформления.
**Цена в USD:** числовое значение цены в долларах.
**Цена в звездах:** числовое значение, отображается рядом со значком звезды.
**Цена в TON:** числовое значение в TON, отображается рядом с соответствующим значком.
**Название подарка:** текстовое поле; краткое название карты.
**URL изображения:** прямая ссылка на изображение, которое будет показано на карточке.
**Количество:** количество подарков. Используется при создании карточки, если нужно показать количество.
**Время:** период действия подарка.

---
 
### Элементы управления
**Сгенерировать карту:** создаёт изображение карточки на основе введённых параметров и показывает его в блоке «Результат».
**История запросов:** открывает список ранее сгенерированных карточек. Позволяет повторно открыть предыдущие результаты.
**Результат:** визуализация итоговой карточки.

---

### Частые ошибки и как их исправить 
**Пустые поля:** заполните обязательные поля (обычно цена, название и URL изображения).
**Неправильный формат цвета:** укажите HEX-цвет вида #RRGGBB. Без дополнительных символов.
**Неверный URL изображения:** убедитесь, что ссылка ведёт непосредственно на файл изображения (расширение .jpg / .png) и доступна публично.
**Необрабатываемое время:** попробуйте указывать время в формате «24h» (часы), «7d» (дни) или «30m» (минуты).
**Картинка не загрузилась:** проверьте доступность по URL и размер изображения.
