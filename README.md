### Описание проекта:

 Данный проект является созданием картинок Telegram gifts

 ---

### Ценность проекта:

Приложение позволяет быстро создавать подарочные карты без навыков дизайна, экономит время - генерирует карты за секунды, все карты в приложении обладают одним стилем
